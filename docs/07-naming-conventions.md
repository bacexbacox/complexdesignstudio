## 7. Konvensi Penamaan

| Jenis | Konvensi | Contoh |
|-------|----------|--------|
| Komponen Svelte | PascalCase | `ServiceCard.svelte`, `HeroSection.svelte` |
| File utility/helper | kebab-case | `slug.ts`, `format-date.ts` |
| File data | kebab-case | `services.ts`, `portfolio.ts` |
| Store / Global State | camelCase + sufiks `State` | `portfolioState.svelte.ts` |
| CSS custom class | kebab-case + prefix `cd-` | `cd-container`, `cd-card` |
| Slug URL | kebab-case lowercase | `website-development`, `name-project-client` |
| Variabel/fungsi TypeScript | camelCase | `getProjectBySlug()`, `buildMeta()` |
| Interface/Type | PascalCase | `PortfolioProject`, `SeoMeta` |
